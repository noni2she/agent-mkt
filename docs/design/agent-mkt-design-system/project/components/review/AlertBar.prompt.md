The console's safety messaging block. `danger` = hard-rule (un-sendable), `warning` = review fatigue / caution, `info` = neutral, `success` = all-clear.

```jsx
<AlertBar tone="danger" title="硬規則命中（不可送出）">不得提供醫療、法律、投資建議</AlertBar>
<AlertBar tone="warning" title="通過率 ≥ 95%，可能審核疲勞">請確認你有逐條讀過原文。</AlertBar>
```
