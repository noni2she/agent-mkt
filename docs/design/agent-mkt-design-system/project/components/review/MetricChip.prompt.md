Threads engagement metric as a mono, tabular pill. Likes keep the native 👍; values are right-readable.

```jsx
<MetricChip kind="likes" value="954" />
<MetricChip kind="replies" value="148" />
<MetricChip kind="age" value="301h" />
```

`kind`: likes / replies / age / followers. These render data straight off `ScoutedPost`.
