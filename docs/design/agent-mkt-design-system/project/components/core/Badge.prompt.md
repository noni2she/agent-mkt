Small quiet label or count pill. Tints by `tone`; `solid` fills it.

```jsx
<Badge tone="brand">reply</Badge>
<Badge tone="success">已送出</Badge>
<Badge tone="neutral" solid>3</Badge>
```

Tones: neutral / brand / success / warning / danger / info. For decision *states* in the review flow, prefer `StatusChip` instead.
