Icon-only square button for toolbars and row actions. Always pass `label` for accessibility.

```jsx
<IconButton icon={<i data-lucide="more-horizontal" />} label="更多" />
<IconButton icon={<i data-lucide="external-link" />} label="開啟原文" variant="solid" />
```

Variants: `ghost` (default), `solid`, `danger`.
